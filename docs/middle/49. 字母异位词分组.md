# 49. 字母异位词分组
给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排列不同的字符串。

示例:

输入: ["eat", "tea", "tan", "ate", "nat", "bat"]
输出:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]

说明：

所有输入均为小写字母。
不考虑答案输出的顺序。
```js
/**
 * @param {string[]} strs
 * @return {string[][]}
 */
var groupAnagrams = function(strs) {
    let map = {}
    for(i in strs){
        let key = strs[i].split("").sort().join("")
        if(!map[key]) {
            map[key] = []
        }
        
        map[key].push(strs[i])
    }

    let rs = []
    for (i in map) {
        rs.push(map[i])
    }

    return rs
};

// [
//     ["ate","eat","tea"],
//     ["nat","tan"],
//     ["bat"]
//   ]
console.log(groupAnagrams(["eat", "tea", "tan", "ate", "nat", "bat"]))
```
